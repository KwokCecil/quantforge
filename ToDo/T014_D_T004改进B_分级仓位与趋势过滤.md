## 三、实施计划

### Step 1：实现分级仓位

将二进制 enter/exit/hold 改为连续的仓位映射：

```python
def _calculate_weight(self, percentile: float) -> float:
    if percentile <= self._config.erp_sell_pct:
        return 0.0
    elif percentile >= self._config.erp_buy_pct:
        return 1.0
    else:
        range_pct = self._config.erp_buy_pct - self._config.erp_sell_pct
        return (percentile - self._config.erp_sell_pct) / range_pct
```

**预期效果**：
- 交易频率降低（阈值附近不再反复切换）
- 回撤控制改善（高估时逐步减仓）
- 心理可执行性提升（不用在 70% 和 69% 之间纠结）

### Step 2：增加趋势过滤器

```python
def _trend_ok(self, close: np.ndarray) -> bool:
    if not self._config.trend_filter_enabled:
        return True
    ma_short = pd.Series(close).rolling(self._config.trend_ma).mean().values
    return close[-1] > ma_short[-1]
```

参数建议：`trend_ma=50`，仅约束买入（卖出不受趋势过滤）。

**注意**：趋势过滤可能与分级仓位有交互效应——需要回测验证是"叠加增强"还是"相互抵消"。
