## 二、目标

**核心诉求**：每次正式回测，自动产出标准化对比图表，不再需要临时写代码。要做到：

1. **单次回测**：BacktestAnalyzer 自动生成标准四面板图（净值+回撤+滚动Sharpe+月度收益热力图），替代当前简陋的双面板
2. **参数扫描**：Validator.parameter_sweep() 跑完后，自动产出一张 Top-N 参数对比图（净值曲线叠加 + 指标柱状图），不再只有 DataFrame
3. **改进对比**（A vs B）：提供统一接口，传入两个 run 目录，自动生成"改进前后 vs 基准"三线对比图
4. **Walk-Forward**：Validator.walk_forward() 跑完后自动产出"样本内外收益瀑布图"

一句话：**图表和数字报告一样，是回测的标准产出，不是事后手工拼凑的**。

## 四、实施计划

### Step 1：创建 `BacktestComparator` 类 ✅

**文件**：`research/backtest_comparator.py`（新建）

这是本次的核心产出——一套可复用的对比可视化工具类。

```python
class BacktestComparator:
    """
    回测结果对比可视化器。
    
    从标准化的 run 目录加载数据 → 生成标准化对比图表。
    所有方法均为静态方法，无状态，纯函数。
    """
    
    # ========== 数据加载 ==========
    
    @staticmethod
    def load_run(run_dir: str) -> dict:
        """加载单个 run 目录。返回 {net_values: DataFrame, analysis: dict, benchmark: Series}"""
    
    @staticmethod
    def load_runs(run_dirs: list[str], labels: list[str] = None) -> dict:
        """批量加载。返回 {labels[i]: {net_values, analysis, benchmark}}"""
    
    # ========== 图表模板（核心） ==========
    
    @staticmethod
    def plot_multi_nv_comparison(
        runs: dict,           # {标签: {net_values, analysis, benchmark}}
        title: str,
        output_path: str,
        figsize=(18, 14),
    ):
        """
        标准四面板对比图：
          Panel ① 净值曲线（多条策略 + 基准填充）
          Panel ② 回撤曲线（多条 fill_between）
          Panel ③ 滚动 Sharpe（126日窗口，零线参考）
          Panel ④ 指标柱状图（总收益/年化/Sharpe/最大回撤，并排柱+数值标注）
        """
    
    @staticmethod
    def plot_ab_comparison(
        run_a: dict,          # 改进前
        run_b: dict,          # 改进后
        title_a: str,
        title_b: str,
        output_path: str,
    ):
        """
        A vs B 改进对比图：
          Panel ① 三线净值（改进前 / 改进后 / 基准）
          Panel ② 回撤对比（两条 fill_between，深浅色）
          Panel ③ 改进前后指标变化（箭头 + 变化量标注）
          Panel ④ 超额收益累计曲线（策略-基准之差）
        """
    
    # ========== 策略特定图表 ==========
    
    @staticmethod
    def plot_timing_signals(
        run: dict,            # 含 signal_log 的策略结果
        signal_fields: list,  # 要绘制的信号字段
        output_path: str,
    ):
        """择时策略专用：价格+买卖点 / 信号指示器 / 净值 / 回撤 —— 四面板"""
    
    @staticmethod
    def plot_param_heatmap(
        sweep_df: pd.DataFrame,  # Validator.parameter_sweep() 的产出
        x_param: str,
        y_param: str,
        z_metric: str = 'sharpe_ratio',
        output_path: str = None,
    ):
        """二维参数热力图，直观展示参数空间的性能地貌"""
    
    # ========== Walk-Forward 专用 ==========
    
    @staticmethod
    def plot_walk_forward(
        wf_df: pd.DataFrame,   # Validator.walk_forward() 的产出
        output_path: str,
    ):
        """Walk-Forward 样本内外收益瀑布图 + 参数漂移时序"""
```
