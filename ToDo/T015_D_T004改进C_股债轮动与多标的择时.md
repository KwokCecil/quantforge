## 三、实施计划

### Step 1：股债轮动 — 对端自动配置

卖出股票后自动配置短债/货基 ETF：

```
enter → target: 510300=100%
exit  → target: 511010=100%  （富国短债ETF，年化~3%）
hold  → 不操作（保持现有仓位）
```

> 注：hold→50/50 股债配比在回测中被证实有害——hold 发生在趋势未确认时，此时减仓股票=在低位割肉。已从最终实现中移除。

**架构改动**：当前 `TimingResolver` 只产出单标的 TargetPosition。需要在 Resolver 层面支持"对端自动配置"——当股票仓位 < 100%，自动将剩余资金配到债基。

```python
class TimingResolver:
    def __init__(self, ..., bond_etf: str = None):
        self.bond_etf = bond_etf  # 对端债基代码

    def resolve(self, decisions, ...):
        targets = []
        for dec in decisions:
            if dec.direction == 'enter':
                targets.append(TargetPosition(code=dec.target_code, weight=1.0))
            elif dec.direction == 'exit' and self.bond_etf:
                targets.append(TargetPosition(code=self.bond_etf, weight=1.0))
            elif dec.direction == 'hold' and self.bond_etf:
                targets.append(TargetPosition(code=dec.target_code, weight=0.5))
                targets.append(TargetPosition(code=self.bond_etf, weight=0.5))
        return targets
```

## 六、备注

- **股债轮动的"轮动"本质**：这不是传统的动量轮动（选最强的），而是"估值驱动"的资产配置切换。框架已经支持，只需要正确的配置
- **多标的择时的资金隔离**：每个标的独立资金池还是共享资金池？建议先用独立池（每个策略独立运行），这是正交设计；后续 T016+ 再做组合级
- **对端债基的选择**：511010（富国短债）流动性好、波动低，适合做现金替代。101年化~3%，相比活期 0.3% 提升约 10 倍
- **与 T016（30年国债趋势择时）区分**：T015 是"用 PE 判断股票贵不贵，贵了买债"，T016 是"用 MA/ROC 判断债券能不能涨，能涨就买债"。两者信号逻辑不同，可以共存
