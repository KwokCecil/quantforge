```python
# research/backtest_engine.py

def run_backtest(strategy, resolver, executor, data_feed, codes, start, end, benchmark_code=None):
    requests = strategy.get_required_data()
    
    # 遍历全部 requests，不只取第 0 个
    all_bar_data = {}
    all_macro_data = {}
    for req in requests:
        resp = data_feed.get_data(req)
        all_bar_data.update(resp.bar_data)
        all_macro_data.update(resp.macro_data)

```

### Step 2：回测循环中逐日分发 macro_data

当前回测循环只分发 `bar_data` 切片。需要同时将 `macro_data` 按日期过滤后注入 `DataResponse`：
