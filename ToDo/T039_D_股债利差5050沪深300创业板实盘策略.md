## 一、策略概要

仅用股债利差信号驱动两个宽基 ETF 的 50/50 买入持有：
- **冲锋信号 → 买入**（510300 50% + 159915 50%）
- **撤退信号 → 清仓**
- **中间持仓期**：任一标的占比 < 33%（不到另一标的一半）→ 触发再平衡回 50/50
- **中性无信号**：如果已持仓则维持，如果空仓则继续等

## 四、实盘设计

### 运行方式

- 每日盘中 Crontab 自动运行 `monitors/guzhai_licha_monitor.py`
- 在 15:05 盘后检查信号
- 状态持久化到 `position/position_guzhai_licha.json`

### 状态文件

```json
{
  "in_market": true,
  "positions": {
    "510300": {"shares": 8900, "cost": 4.559},
    "159915": {"shares": 14400, "cost": 2.822}
  },
  "last_signal": "charge",
  "last_update": "2026-05-21"
}
```

### 通知内容

- 冲锋：`【股债利差】冲锋信号触发！建议买入 510300(50%) + 159915(50%) 当前分位=XX%`
- 撤退：`【股债利差】撤退信号触发！建议清仓 510300 + 159915 当前分位=XX%`
- 再平衡：`【股债利差】再平衡信号！{标的}占比{X%}，建议调整回50/50`
- 每日状态：发送当前持仓和净值
