## 二、目标

实现一个简单的趋势跟踪择时策略，在 30 年国债 ETF（511010）上用 MA/ROC 指标判断趋势：
- 上升趋势 → enter（全仓）
- 下降趋势 → exit（清仓）
- 震荡 → hold（观望）

策略逻辑简单、数据已有、验证快速。目标是验证"非股票类标的 + TIMING 策略"的框架通用性。

### Step 1：实现 TreasuryTrendConfig

```python
@dataclass
class TreasuryTrendConfig(StrategyConfig):
    """30年国债趋势择时配置。"""

    # === 核心参数 ===
    etf_code: str = "511010"          # 30年国债 ETF
    trend_ma_short: int = 20          # 短均线周期
    trend_ma_long: int = 60           # 长均线周期

    # === 信号阈值 ===
    roc_period: int = 10              # ROC 回看期
    roc_buy_threshold: float = 0.005  # ROC > 0.5% → 看多
    roc_sell_threshold: float = -0.003 # ROC < -0.3% → 看空

    # === 风控 ===
    stop_loss: float = 0.03           # 成本止损 3%
    high_water_stop: float = 0.05     # 高水位止损 5%
```

### Step 2：实现 TreasuryTrendStrategy

```python
class TreasuryTrendStrategy(Strategy):
    """30年国债趋势择时。双均线判断趋势 + ROC 确认信号 → TIMING Decision。"""

    def produce_decisions(self, data: DataResponse) -> list[Decision]:
        close = data.bar_data[self._config.etf_code]['close']

        # 趋势判断：短均在长均之上
        ma_short = pd.Series(close).rolling(self._config.trend_ma_short).mean()
        ma_long = pd.Series(close).rolling(self._config.trend_ma_long).mean()
        trend_up = ma_short.iloc[-1] > ma_long.iloc[-1]

        # 动量确认
        roc = (close[-1] / close[-self._config.roc_period]) - 1

        # 决策
        if trend_up and roc > self._config.roc_buy_threshold:
            decision_type = DecisionType.TIMING
            direction = 'enter'
        elif not trend_up and roc < self._config.roc_sell_threshold:
            decision_type = DecisionType.TIMING
            direction = 'exit'
        else:
            decision_type = DecisionType.TIMING
            direction = 'hold'

        return [Decision(...)]
```

## 六、备注

- **国债 ETF 的特征**：511010 的波动远小于股票 ETF（年化波动 ~3% vs ~25%），Sharpe 天然更高
- **趋势策略在债券上的有效性**：债券是趋势性较强的资产（利率周期通常持续 1-3 年），趋势跟踪在债券上理论上比在股票上更有效
- **利率环境的转变**：2025-2026 年如果利率触底反弹（债市转熊），趋势策略能否及时 exit 是核心考验
- **与其他策略的关系**：T016 是独立策略，与 T004（股债利差择时，择的是股票）不冲突，可以对端运行
