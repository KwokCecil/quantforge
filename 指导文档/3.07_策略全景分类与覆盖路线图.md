## 一、经典量化策略全景

量化策略按**赚什么钱**分成 7 大类。下表是理论全景：

### 1. 趋势/动量类（Trend & Momentum）

赚"强者恒强、弱者恒弱"，行为金融学来源：投资者反应不足、羊群效应。

| 子类 | 代表策略 | 典型参数 |
|------|----------|----------|
| 时间序列动量 | ROC(Price)、均线交叉（SMA/EMA）、MACD | lookback=15~252 |
| 截面动量 | 行业内/跨资产选强，Jegadeesh-Titman 经典 | top_k=3~10 |
| 多周期动量 | 短/中/长周期信号加权融合 | 3~5 个周期加权 |
| 残差动量 | 剥离市场beta后的特质收益排序 | OLS回归残差 |
| 波动率加权 | 仓位 ∝ 1/vol，降低高波动标的暴露 | 滚动20~60日 |

### 2. 反转/均值回归类（Mean Reversion）

赚"涨多了会跌，跌多了会涨"，过度反应后的价格修复。

| 子类 | 代表策略 | 典型参数 |
|------|----------|----------|
| 短期反转 | 1~5日间隔的反转效应 | reversal_window=3~5 |
| 波幅反转 | RSI超买超卖、布林带边界回归 | RSI(14) > 70 / < 30 |
| 统计套利 | 配对交易（cointegration）、三角套利 | 需做空机制 |

### 3. 因子/多因子类（Factor Investing）

赚"承担系统风险换取收益"，Fama-French 体系的扩展。

| 因子 | 含义 | 典型指标 |
|------|------|----------|
| 市场 Beta | 市场系统性风险敞口 | CAPM β |
| 规模 SMB | 小盘股超额收益 | 市值分层 |
| 价值 HML | 低估值（高B/P）超额 | PE、PB、股息率 |
| 动量 MOM | 过去赢家继续赢 | 12-1 动量 |
| 质量 QMJ | 高ROE/低杠杆/低应计 | ROE、资产负债率 |
| 低波动/低Beta | 低波动反常跑赢 | 历史波动率、β |
| 流动性 | 低流动性溢价 | 换手率、Amihud |

### 4. 套利类（Arbitrage）

赚定价偏差，理论上近乎零风险。

| 子类 | 代表策略 | 数据依赖 |
|------|----------|----------|
| ETF套利 | 折溢价套利（IOPV vs 市价）、申赎套利 | 集思录溢价率 |
| 期现套利 | 股指期货 vs 现货基差套利 | 期货行情 |
| 跨期/跨市场 | 同一品种远近月、不同交易所价差 | 多市场数据 |
| 转债套利 | 可转债 vs 正股 | 转债行情 |

### 5. 择时/宏观类（Market Timing）

赚"什么时候买比买什么更重要"。

| 子类 | 代表策略 | 数据源 |
|------|----------|--------|
| 股债轮动 | 估值分位数切换股/债/现金 | PE、国债收益率 |
| 行业轮动 | 经济周期切换行业配置 | 宏观指标、行业ETF |
| 市场情绪 | 恐慌指数(VIX)、融资融券余额 | 情绪数据源 |
| 资金流 | 北向资金、机构净买入 | 资金流数据 |

### 6. 事件驱动（Event Driven）

赚"信息冲击后的价格偏移"。

| 子类 | 代表策略 | 数据源 |
|------|----------|--------|
| 财报效应 | 业绩预增/预减公告后漂移(PEAD) | 财报公告 |
| 并购套利 | 被收购方 vs 收购方价差 | 并购公告 |
| 分红/送转 | 除权除息日前后异常收益 | 分红公告 |

### 7. 另类/高频（Alternative / HFT）

赚另类数据或执行速度。

| 子类 | 代表策略 | 数据源 |
|------|----------|--------|
| 舆情/NLP | 新闻情感打分选股 | 文本数据、NLP |
| 高频做市 | 买卖价差 | Level 2 行情 |
| 分钟K/日内 | T+0 日内趋势/反转 | 分钟K线 |

### 2.2 ROCStrategy 多模式引擎分析（已全面验证）

`ROCStrategy` 不是一个单纯的动量轮动策略——它通过配置开关内聚了 17 种开关，是一台**可插拔的多模式动量引擎**。经 T025/T028/T033 全周期回测（2018-2026），结论如下：

| 模式 | 触发开关 | 结论 | 说明 |
|------|----------|------|------|
| 单因子ROC | （默认） | ✅ **Baseline** | ROC(22) + tech_growth 33只池，裸ROC全周期年化-8.3%/回撤83% |
| 波动率倒数加权 | `inverse_vol_weight=True` | ✅ **必开** | 实证最优：降低高波动标的暴露 |
| ATR+ADX趋势择时 | ATR+ADX（策略开关） | ✅ **必开** | 替代死叉择时，T025全周期年化+11.1% |
| 信号过滤 | `rsi_filter_enabled`（RSI≥60禁买） | ✅ **必开** | T028验证，回撤-20pp |
| 宏观阻断 | `macro_block_enabled`（ATR/ADX代理510300） | ✅ **必开** | T033验证，Sharpe 0.63→1.19 |
| 时序动量过滤 | `ts_momentum_enabled=True` | ❌ 不建议 | 全周期熊市帮倒忙（-14.4%），阻止抄底反弹 |
| 多因子ROC | `multi_factor=True` | ❌ 不建议 | 因子共线导致收益-12.2% |
| 投票增强 | `voting_enabled=True` | ❌ 不建议 | 因子共线（>0.83），投票无效 |
| RSI替代 | `primary_factor="RSI"` | ❌ 不建议 | 全周期0笔交易触发（结构性劣势） |
| 残差动量 | `residual_momentum_enabled=True` | ❌ 不建议 | 无效（T021+T025结论一致） |
| 崩盘防护 | `crash_protection_enabled=True` | ❌ 不建议 | 降低回撤但牺牲收益 |

**进化路径**（统一口径：2018-01-01 ~ 2026-05-21，基准创业板399006）：
```
T024 裸ROC                  年化 -8.3%     回撤 83%
T025 +池子+攻防(ATR/ADX)    年化 +11.1%    Sharpe 0.63
T028 +信号过滤(RSI≥60禁买)  回撤 -20pp
T033 +宏观阻断(ATR/ADX代理)  Sharpe 0.63→1.19
截止2026-05-21持仓自然增值   年化 14.71%→17.09%
= tech_growth              年化 +17.09%  Sharpe 1.32  DD 14.22%
```

**最终生产配置**（tech_growth.json）：8 开关开（ROC(22) + inverse_vol + ATR/ADX择时 + RSI≥60禁买 + 宏观阻断 + top_k=5 + buy_roc_edge=20.0 + 33只科技/成长池），其余开关关。

> 开关结论的 SSOT 在 [5.08_策略开关有效性研究总表.md](./5.08_策略开关有效性研究总表.md)，最新回测数据在 [5.12_现阶段成果总结.md](./5.12_现阶段成果总结.md)。
