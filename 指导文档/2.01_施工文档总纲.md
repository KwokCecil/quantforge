# QuantForge L3 施工文档

> 本文档基于《1.01_工程规格书》和《1.02_演进策略》制定，是逐步实施的详细施工指南。

---

## 总体路线

```
Phase 1 ──→ Phase 2 ──→ Phase 3 ──→ Phase 4
核心骨架      验证基建     多策略+实盘    系统完善
+ ROC落地     FactorLab   TimingResolver  可视化
              Validator   爬虫+通知+实盘   摩擦分析
```

每个 Phase 完成后都是一个**可工作的系统**，不是半成品。

---

## 技术约束

| 项目 | 约束 |
|------|------|
| Python 版本 | 3.11+ |
| 编码风格 | 不添加注释（除非明确要求） |
| 敏感信息 | 环境变量 + .env 文件，不入版本控制 |
| 交易规则 | A股：100份整数倍，佣金万2.5最低5元，T+1 |
| 数据格式 | DataFrame 标准列：date/open/close/high/low/vol |
| 日志 | loguru，INFO+ERROR 双文件，监控进程按日期额外日志 |
| 扩展原则 | L3 只实现规格书定义的功能，不提前设计 L3.5+ |

## 风险提示

1. **API 依赖**：autostock.cn API 需要 token，需确保 .env 配置正确
2. **数据质量**：ffill 引入偏差（停牌日填充），需在数据质量检查中标记
3. **回测一致性**：Phase 1 完成标志是 ROC 回测结果正确，需仔细验证
4. **实盘安全**：LiveExecutor 不自动下单，只推送建议；持仓持久化需异常保护
