# 3.05 标的筛选的过拟合风险分析

> 生成时间：2026-05-05
> 讨论问题：先单独回测每只标的在动量策略下的表现，剔除表现差的，保留"有效"标的组成标的池。这种做法是否有过拟合风险？筛选后的标的池是否一定优于全量标的池？

---

## 一、核心判断：存在严重的过拟合风险

**结论先行：这种"先筛选后回测"的做法存在显著的多重过拟合风险，绝对不能保证筛选后的标的池在未来的表现一定好于未经挑选的标的组合。**

这不是一个模棱两可的灰色地带问题，而是一个在量化研究中已经被充分讨论和反复验证的经典陷阱。

---

## 二、你实际上做了什么——梳理问题本质

流程可以抽象为：

```
Step 1: 对 N 只标的逐个运行动量策略 → 得到 N 条净值曲线
Step 2: 按某个绩效指标（Sharpe、收益、回撤等）排序
Step 3: 剔除表现差的标的 → 保留 M 只"有效"标的
Step 4: 用这 M 只标的重新回测（或直接声称这个标的池更好）
```

核心问题：**Step 2~3 的筛选决策使用了未来的信息**。当你用全量历史数据回测单标的时，你已经"看到了"每只标的在整个回测区间内的表现，然后据此决定哪些标的该被留在池子里。

---

## 三、过拟合的具体机制

### 3.1 幸存者偏差（Survivorship Bias）

你筛选出的 M 只标的之所以表现好，很可能只是因为：

- 它们在**这段特定的历史区间内**恰好趋势流畅、动量效应显著
- 这种"流畅"可能来自那段时间的特定宏观环境、行业轮动节奏、或者纯粹的随机漫步
- **你看的是已经发生的历史结果，然后选择性地保留了"赢家"**

一个形象的类比：先看完了整场足球比赛的录像，然后再选择哪些球员"表现好"来组建你的梦幻球队。你对这支球队的预期表现会远远高估。

### 3.2 数据窥探偏差（Data Snooping Bias）

这个问题可以形式化地描述。假设：

- 你有 N = 32 只候选标的（当前 config 文件中的数量）
- 每只标的的历史表现 = 真实动量效应 + 随机噪声
- 即使动量效应完全不存在（纯随机游走），由于随机性，大约一半的标的在回测中会显示正收益

当你**在同一个数据集上**先筛选后回测时：
- 筛选标准捕获了随机噪声中"看起来像动量"的部分
- 组合在一起后，这些噪声可能相互抵消，也可能叠加放大
- 但无论如何，回测结果都**不可靠地偏高**了

### 3.3 多次比较问题（Multiple Comparison Problem）

从统计学角度，你进行了 N 次独立测试（N = 标的数量），然后选择了表现最好的子集。这就好比：

> 抛 32 次硬币，每次都抛 1000 天，然后选出那些正面偏多的硬币，声称它们是"有偏的硬币"。

即使所有硬币都是公平的，你总能选出一些"看起来有偏"的硬币——这就是选择性报告。

### 3.4 参数与标的的联合过拟合

ROC 策略本身已经有很多可调参数：

| 参数类别 | 参数 | 说明 |
|---------|------|------|
| 核心参数 | `buy_roc_edge`, `roc_n`, `roc_m` | 直接影响买入时机 |
| 辅助参数 | `ma_period`, `sell_ma_roc_edge` | 影响过滤条件 |
| 条件开关 | `STRICT_BUY`, `MA_PRICE_CROSS`, `CUT_LOSS` 等 | 多个二元开关 |
| 止损参数 | `cut_loss_edge`, `high_watermark_stop_edge` | 影响出场 |

**参数优化 + 标的筛选的组合爆炸效应**：

如果参数空间有 K 种组合，标的有 N 只，你要从 N 选 M 组成标的池:
- 你同时探索了 K × C(N, M) 种可能的配置
- 从中选出了表现最好的那一种
- **这在统计学上相当于对历史数据做了极度激进的曲线拟合**

## 四、为什么筛选后的标的池不一定优于全量池？

### 4.1 均值回复的力量

在金融市场中，过去表现好的标的/策略往往面临均值回复：

- **行业轮动**：过去 3 年动量强的行业，未来可能因为估值过高、资金撤离而表现逆转
- **因子拥挤**：当某个因子在特定标的上被证明有效后，资金涌入会使得该因子的超额收益被稀释甚至转为负值
- **结构性变化**：市场微观结构、监管政策、宏观经济环境的变化，都会使得"过去有效"的模式在未来失效

你筛选出的标的池天然偏向于**最近趋势流畅**的品种，而这恰恰是未来最容易均值回复的品种。

### 4.2 集中度风险的隐性代价

当前 config 的默认标的池有 32 只 ETF，涵盖通信、消费、医药、白酒、创业板、科创板、半导体、光伏、稀土、化工、证券、新能车等多个行业和风格。

你筛选后保留的"有效"标的，很可能在行业或风格上高度集中（比如偏向科技/成长风格），因为：
- 过去一段时间趋势流畅的标的往往是同一风格
- 动量策略天然会在趋势强的标的上产生好结果

这种**隐性风格集中**在筛选区间表现好，但在风格切换时会遭受毁灭性打击。

### 4.3 样本量的矛盾

由于 A 股 ETF 的历史普遍不长（很多 2019 年以后才成立），你的回测区间很可能只有 3~5 年。这么短的时间窗口：
- 不足以覆盖完整的牛熊周期
- 不足以在统计上区分"真的动量效应"和"随机的好运气"
- 筛选出的标的池更像是"在这段特定历史上表现好的集合"而非"真正有动量效应的集合"

### 4.4 单标的与组合回测的本质差异

单独回测每只标的时，策略 100% 的时间只关注这一只标的。但组合回测时：
- 资金要在多只标的中分配
- 买卖信号之间存在交互（资金竞争、优先级排序、调仓时机交错）
- 单标的表现好 ≠ 它在组合中也对整体表现有正面贡献

## 五、正确的做法

### 5.1 时间序列交叉验证（Walk-Forward Analysis）

将数据按时间切分：

```
训练期 (2020-2022)  →  验证期 (2023-2024)  →  测试期 (2025+)
```

- 在**训练期**做标的筛选和参数优化
- 在**验证期**评估标的池的稳健性（可以调整方案）
- 在**测试期**做最终评估，**绝不再回头修改任何决策**

关键原则：**任何涉及选择的决策（选标的、选参数），都必须在时间上严格"使用过去预测未来"。**

### 5.2 Out-of-Sample 严格隔离

如果你想筛选标的池：

```
Step 1: 用 2020-2022 的数据，筛选出 M 只标的
Step 2: 用 2023-2025 的数据，回测这个 M 标的池 → 这才是"真实可预期的表现"
```

如果你在 2023-2025 的数据上筛选，又在同一段数据上回测 → **这没有任何预测价值。**

### 5.3 不要过度筛选，考虑更稳健的方案

| 方案 | 优点 | 缺点 |
|------|------|------|
| **全量标的池**（全部参与） | 无过拟合风险；分散化好 | 可能包含了动量效应不明显的标的 |
| **基于逻辑筛选**（流动性、规模、成立时间） | 逻辑透明，不过拟合 | 不保证绩效提升 |
| **等权/分层**（行业均衡的标的池） | 降低风格集中风险 | 复杂度增加 |
| **动态标的池**（基于近期动量排名动态选择） | 适应市场变化 | 交易频率较高 |

实际上，你当前策略中已经有的机制——**按 ROC 值排序，动态选择 top_k 只标的**——本质上就是一种实时、无过拟合的标的筛选机制。在策略运行时，根据当天的动量值决定买什么，这比根据历史回测结果"一劳永逸"地固定标的池要好得多。

### 5.4 对现有标的池做稳健性检验

如果你想验证当前标的池是否真实优于全量池，可以进行以下测试：

1. **随机子集检验**：从 N 只标的中随机抽取 M 只（M 等于你当前标的池大小），重复 100 次，看你的标的池表现排在什么分位数
2. **时间段拆分检验**：将回测区间均匀拆成 3~4 段，看标的池在每一段是否都优于全量池
3. **行业分层检验**：按行业分组，看筛选后的标的池是否在行业分布上与全量池存在系统性偏差

## 六、小结

| 问题 | 答案 |
|------|------|
| 有没过拟合风险？ | **有，且非常严重。** 这是量化研究中经典的 data snooping 问题 |
| 标的池是否一定优于全量池？ | **不能保证。** 筛选后的标的池的未来表现很可能低于其回测表现，甚至低于全量池 |
| 最核心的错误在哪里？ | **在同一个数据集上既做筛选决策又做绩效评估**，违反了 out-of-sample 验证的基本原则 |
| 应该怎么做？ | 采用 Walk-Forward 时间隔离；或信任策略自身的动态标的选择机制 |

**一句话总结：你筛选的不是"真正有效的标的"，而是"在这段历史数据上恰好表现好的标的"。这两者之间的差距，就是过拟合的全部代价。**
